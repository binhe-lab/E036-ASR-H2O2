2, Scan using: scroll down the bar, choose Motif Set

3, Select TF Set: Expert Curated - no dubious

4, Enter DNA sequence: 1kb 5'UTR of CgCTA1

Then it will give u the TFs prediction output (based on S. cerevisiae)
